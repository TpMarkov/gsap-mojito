**Project: Mojito (Animated Portfolio)**
Designed and built a high-performance interactive portfolio using **React 19** and **Vite 7**. Implemented complex, timeline-based animations with **GSAP** and developed a responsive, modern UI with **Tailwind CSS 4**. The project demonstrates mastery of advanced frontend techniques, ensuring 60fps animations and a seamless user experience across all devices.
*Technologies: React, GSAP, Tailwind CSS, Vite.*
